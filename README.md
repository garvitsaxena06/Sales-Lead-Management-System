# POC - Sales-Lead-Management-System

> Web app created using Node.js, Express.js, PostgreSQL, and Bootstrap.
> CRUD application to collect sales leads (also includes user auth and admin login).

## Install Dependencies
```
npm install
```

## Run App
```
# Run in dev mode
npm run dev

#Run in prod mode
npm start
```
